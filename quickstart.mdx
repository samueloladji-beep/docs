## 1. Install the SDK
```bash
pip install vaultak
```

## 2. Get your API key

Sign up at [app.vaultak.com](https://app.vaultak.com). Your API key is shown once on the onboarding screen — save it.

## 3. Wrap your agent
```python
from vaultak import Vaultak

vt = Vaultak(api_key="vtk_your_api_key_here")

with vt.monitor("my-agent"):
    # your agent code here
    pass
```

That's it. Every action your agent takes is now monitored, risk-scored, and logged in real time on your dashboard.

## Response Modes

| Mode | What happens |
|------|-------------|
| `ALERT` | Logs and notifies. Agent keeps running. Default. |
| `PAUSE` | Agent halts. Awaits human approval. |
| `ROLLBACK` | Reverses last N actions. Agent pauses. |

## Pre-execution checks
```python
import requests

response = requests.post("https://vaultak.com/api/check",
    headers={"x-api-key": "vtk_your_api_key_here"},
    json={
        "agent_id": "my-agent",
        "action_type": "file_delete",
        "resource": "/data/sensitive.csv"
    }
)

result = response.json()
if result["allowed"]:
    delete_file("/data/sensitive.csv")
else:
    print(f"Blocked: {result['reason']}")
```

## Works with any framework
```python
# LangChain
with vt.monitor("langchain-agent"):
    response = llm.invoke(prompt)

# CrewAI
with vt.monitor("crew-agent"):
    crew.kickoff()

# Custom agents
with vt.monitor("my-agent"):
    agent.run()
```
