This guide is for developers who want to add Vaultak directly to their agent code. If you want to govern an existing agent without code changes, see the [Vaultak Sentry guide](/sentry).

## Step 1: Install

```bash
pip install vaultak
```

## Step 2: Get your API key

Sign up at [app.vaultak.com](https://app.vaultak.com). Your API key is shown once on the onboarding screen.

## Step 3: Wrap your agent

```python
from vaultak import Vaultak

vt = Vaultak(api_key="vtk_your_api_key_here")

with vt.monitor("my-agent"):
    # your agent code here
    pass
```

That is it. Every action your agent takes is now monitored, risk-scored, and logged in real time.

## Step 4: Define what your agent is allowed to do

```python
from vaultak import Vaultak, ActionType, KillSwitchMode

vt = Vaultak(
    api_key="vtk_your_api_key_here",
    allowed_action_types=[ActionType.FILE_READ, ActionType.API_CALL],
    allowed_resources=["/tmp/*", "/data/readonly/*"],
    blocked_resources=["prod.*", "*.env"],
    max_actions_per_minute=20,
    max_risk_score=0.7,
    mode=KillSwitchMode.PAUSE
)

with vt.monitor("my-agent"):
    agent.run()
```

## Response modes

ALERT: Logs and notifies. Agent keeps running. Default.
PAUSE: Agent halts immediately. Awaits human approval in dashboard.
ROLLBACK: Reverses last N actions automatically. Agent then pauses.

## Works with any framework

```python
# LangChain
with vt.monitor("langchain-agent"):
    response = llm.invoke(prompt)

# CrewAI
with vt.monitor("crew-agent"):
    crew.kickoff()

# AutoGen
with vt.monitor("autogen-agent"):
    agent.run()
```
