## Installation

Install the SDK with pip:

    pip install vaultak

## Initialization

    from vaultak import Vaultak
    vt = Vaultak(api_key="vtk_your_api_key_here")

## Parameters

- api_key (required) - Your Vaultak API key from the dashboard
- mode - Response mode: ALERT, PAUSE, or ROLLBACK. Default: ALERT
- risk_threshold - Float 0-1 that triggers the response. Default: 0.7
- rollback_limit - Max actions to reverse in ROLLBACK mode. Default: 5

## vt.monitor()

Context manager that wraps your entire agent session.

    with vt.monitor("my-agent"):
        pass

## vt.log_action()

Explicitly log a single agent action.

    vt.log_action(agent_id="my-agent", action_type="file_delete", resource="/data/sensitive.csv")

## 5-Dimensional Risk Scoring

- Action severity - How destructive is this action type?
- Resource sensitivity - How sensitive is the target resource?
- Payload anomaly - Does the payload deviate from baseline?
- Frequency - Is the agent acting unusually fast?
- Context - Does this action fit the agents normal pattern?

## REST API

All endpoints require: x-api-key: vtk_your_api_key_here

POST /api/check - Pre-execution risk check
POST /api/actions - Log a completed action
GET /api/agents - List all registered agents
GET /api/alerts - Retrieve active alerts
GET /api/policies - List security policies
POST /api/policies - Create a security policy
DELETE /api/policies/ID - Delete a policy
