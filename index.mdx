## What is Vaultak?

Vaultak is a runtime security platform for autonomous AI agents. It monitors every action your agents take, enforces the boundaries you define, and automatically responds when something goes wrong.

## Two ways to get started

### Vaultak Core

For developers who want deep code-level integration. Add five lines of code to your agent and get full behavioral monitoring, risk scoring, permission profiles, and automatic rollback.

```bash
pip install vaultak
```

### Vaultak Sentry

For teams who want to govern existing agents without touching the code. No code changes. Works with any language and any framework.

```bash
pip install vaultak-sentry
```

Both products use the same API key and the same dashboard at app.vaultak.com.

## What makes Vaultak different?

- **Zero code changes with Sentry** - govern any agent without modifying it
- **Automatic rollback** - reverses the last N agent actions on violation. No other tool does this.
- **Policy compiler** - declare exactly what each agent is authorized to do
- **Pre-execution enforcement** - actions are blocked before they run, not just logged after
- **Three response modes** - Alert, Pause, and Rollback
- **HIPAA and SOC 2 ready** - full audit trail for compliance documentation
